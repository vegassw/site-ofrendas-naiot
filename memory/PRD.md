# PRD - Naiot Ministerio Cristiano Landing Page

## Fecha: Febrero 2026

## Problem Statement Original
Landing page responsiva y atractiva para el Ministerio Naiot que muestre los datos de transferencia bancaria para ofrendas y diezmos. Incluye botones flotantes de redes sociales (Instagram, Facebook, TikTok).

## User Personas
- Miembros del Ministerio Naiot que desean realizar ofrendas/diezmos por transferencia
- Visitantes que buscan información de contacto y redes sociales del ministerio

## Core Requirements (Static)
1. Página única con datos de transferencia bancaria
2. Logo del ministerio visible
3. Botones flotantes de redes sociales
4. Diseño responsivo (móvil y desktop)
5. Funcionalidad de copiar datos al portapapeles
6. Estilo mixto (sección oscura hero + sección clara contenido)
